## Brand quick reference

**Colours**

| Role | Hex |
|---|---|
| Deep Orange (primary dark — left half of shield) | `#D2451D` |
| Bright Orange (primary — right half of shield) | `#EC6B1D` |
| Highlight Orange (accents/gradients) | `#F56F1E` |
| Wordmark Grey ("SecureVisa Group" lettering) | `#757575` |
| Signature Blue (names/links in email signatures) | `#3070D1` |
| Signature Orange (dividers/web links in signatures) | `#F76B36` |
| Body Text | `#444444` |
| Legal Grey (email disclaimers) | `#B2B2B2` |

Exact Pantone print codes: `Pantone Code SVG.pdf` in the Logo Packs folder.

**Typography** — *Friska* for headlines and artwork (install it before opening
any `.ai`/`.psd`); *Arial* for email signatures and HTML business cards.

## First-day checklist for the new owner

1. Get access to the [Marketing Digital Assets](https://itsecae.sharepoint.com/sites/MarketingDigitalAssets) SharePoint site.
2. Bookmark the brand-guide links above (claude.ai share access needed).
3. Install the **Friska** font from *Logo Packs → SecureVisa Font*.
4. Make sure Adobe Illustrator + Photoshop are available (masters are `.ai`/`.psd`).
5. Open `index.html` and skim the asset library and how-to playbooks.

## Keeping this handover useful

When assets move or are added on SharePoint, update the matching card in
`index.html` (and this README) and commit. Phase 2 will add the ITSEC brand
alongside this one.
