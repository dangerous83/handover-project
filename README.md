# SecureVisa Group — Marketing & Design Handover

**July 2026** — two brand handovers in one repo: **SecureVisa** ([`index.html`](index.html))
and **ITSEC** ([`itsec.html`](itsec.html)). Each brand has its own page so there's no confusion
about which one you're working on.

Welcome! Open **[`index.html`](index.html)** in any browser — it's a single, self-contained
page (all images embedded) that maps out every brand asset, where it lives on
SharePoint, and how to handle the most common requests. The **ITSEC** brand has its own
matching page at **[`itsec.html`](itsec.html)**.

## First-day checklist for the new owner

1. **Get access to the ITSEC SharePoint** — everything lives under [Marketing Digital Assets → SVG](https://itsecae.sharepoint.com/:f:/r/sites/MarketingDigitalAssets/Shared%20Documents/SVG?csf=1&web=1&e=DKNOFT). Request access from IT if it asks you to sign in.
2. Bookmark the brand-guide links above (claude.ai share access needed).
3. Install the **Friska** font from *Logo Packs → SecureVisa Font*.
4. Make sure Adobe Illustrator + Photoshop are available (masters are `.ai`/`.psd`).
5. Open `index.html` and skim the asset library and how-to playbooks.

## Keeping this handover useful

When assets move or are added on SharePoint, update the matching card in
`index.html` / `itsec.html` (and this README) and commit. Other group brands
can each get their own page the same way ITSEC did.
