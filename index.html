How do I…? The four most common requests

Step-by-step recipes for the jobs that will land on your desk most often.

👋 Set up a new joiner

  1. Email signature: in the Email Signature folder, duplicate the closest existing HTML file (e.g. Cathyrene Tan SVG Email Signature.html) and rename it to the new person.
  2. Edit the file: name & job title (the blue block), then the E / M / T contact lines. Everything else stays as-is.
  3. Keep the img/ folder next to the HTML — the logo GIF loads from there.
  4. Open the HTML in a browser, select-all, copy, and paste into Outlook’s signature editor.
  5. Business card: update a copy of the latest card master with their details and export a print PDF; add a digital-card JPG if they need one for WhatsApp/email.

🖃 Create a client stamp

  1. Open 1-EDITABLE STAMP.pdf in the Stamp SVG folder.
  2. Replace the company name with the client’s full legal name (as licensed).
  3. Export as PNG at the same size as the existing examples.
  4. Save it into Clients Stamp/ named exactly after the client, e.g. VERDAD PROJECT MANAGEMENT SERVICES CO. L.L.C.png.

📊 Build a proposal structure diagram

  1. Start from Diagaram Proposal.pdf in the Diagram Structure folder.
  2. Adapt the entity boxes to the client’s proposed setup — if two options are on the table, label them A and B like the NUR TAKSON example.
  3. Stay in brand: oranges for SecureVisa entities, grey for third parties.
  4. Save the approved version into FA/ as CLIENT NAME – A/B.pdf.

📄 Issue a letterhead document

  1. Make sure Friska is installed (see the checklist above).
  2. Open SVG Letterhead.ai from the Letterhead folder.
  3. Compare against SVG Letterhead - 2025-01.jpg — that’s the approved look.
  4. Export as PDF; for Word-based letters, drop the letterhead in as a full-page background.
